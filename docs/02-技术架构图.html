Technical Architecture

人生平行宇宙实验室 · 技术架构图

基于 Vue 3 + Express.js + DeepSeek AI 构建的全栈智能推演平台，
通过分层架构实现前端展示、接口服务、AI 推演与数据持久化的解耦协作。

🎨
Layer 01 · Presentation
前端展示层
1
Vue 3 (Composition API) Vite Vue Router ECharts html2canvas + jsPDF

负责用户界面渲染与交互体验。采用 响应式设计 适配多终端， 以 玻璃拟态 UI 营造科技氛围，配合 动画系统 呈现平行宇宙的推演过程，并支持将推演结果一键导出为图片或 PDF。

Layer 02 · API Gateway
API 接口层
2
Express.js RESTful API Axios CORS 15+ 端点

作为前后端通信枢纽，提供 15+ RESTful 端点 统一调度业务逻辑。 内置 统一错误处理 机制保障服务稳定性，并具备 Mock 数据兜底 能力，在 AI 服务不可用时仍可正常运行。

架构数据流： 用户在 前端展示层 发起操作，经由 Axios 通过 API 接口层 转发请求；接口层根据业务类型分别调度 AI 推演层（调用 DeepSeek 生成推演结果）与 数据层（读写 MySQL 与本地存储），最终将聚合结果回传前端渲染呈现。 三层架构职责清晰、松耦合，便于独立扩展与维护。
