1 开场 · 演示模式 0:00 — 0:30
01操作步骤
  1. 打开 Chrome 浏览器，在地址栏输入 http://localhost:5173 访问项目首页。
  2. 等待首页加载完成，目光定位顶部导航栏右侧的"演示模式"按钮。
  3. 点击"演示模式"按钮，页面路由跳转至 /demo 演示模式页面。
  4. 停留于封面页，等待全屏渐变标题动画播放完毕。
02解说词

各位评委老师好，接下来由我为大家现场演示"人生平行宇宙实验室"。这是一款用 AI 帮你预见不同选择下人生轨迹的决策成长平台。

我们专门为今天的评审准备了一个演示模式，它会用全屏滚动的方式，把项目的价值、创新、架构和应用场景一次性呈现给大家。请各位先看封面——

2 演示模式滚动展示 0:30 — 1:30
01操作步骤
  1. 在 /demo 页面缓慢向下滚动鼠标滚轮，逐屏推进。
  2. 依次经过四个全屏 Section：项目价值 → 核心创新 → 技术架构 → 应用场景。
  3. 每个 Section 滚动吸附到位后停留 3-5 秒，等待入场动画完成再继续下滚。
  4. 滚动至最后一屏后，点击底部"立即体验"按钮，准备进入核心功能。
02解说词

现在向下滚动，第一屏是项目价值——我们瞄准的是 87% 年轻人面临的决策焦虑，让"看不见的未来"变成"看得见、比得了、能复盘"的平行宇宙。

第二屏是核心创新——四重平行宇宙推演，AI 一次生成四条人生路径，每条都有完整的未来画像。

第三屏是技术架构——前端 Vue3 + ECharts，后端 Node.js + MySQL，AI 推理层负责生成多维度决策分析。

最后一屏是应用场景——覆盖职业选择、城市去留、考研就业、跳槽转型等高频决策场景。下面我们进入最核心的功能演示。

  2. 点击底部"启动平行宇宙模拟"按钮，触发 AI 推演流程。
02解说词

现在进入核心功能。我们以一位 28 岁的北京前端工程师为例，他月薪一万五、存款十万，正面临一个纠结——要不要辞职创业做独立开发者。

这些参数不是随便填的：年龄和学历决定职业天花板，城市影响机会成本，收入和存款决定抗风险能力。把真实的人生参数交给 AI，它才能推演出贴合实际的未来。

点击"启动平行宇宙模拟"——请注意看加载动画，AI 正在为这个决定推演四条不同的未来。

4 结果展示 · 四重宇宙 2:00 — 2:45
01操作步骤
  1. 等待结果页加载完成，四个宇宙卡片依次渐入。
  2. 依次介绍四张卡片，鼠标悬停高亮当前卡片：
宇宙A · 安稳现状 宇宙B · 突破执行 宇宙C · 成长折中 宇宙D · 创新路径
  1. 点击"宇宙B · 突破执行"卡片，展开查看详情：未来图景、十年时间线、收入变化曲线、幸福指数。
  2. 再点击"宇宙D · 创新路径"卡片，与宇宙B 形成对比展示。
02解说词

推演完成。AI 为这个决定生成了四个平行宇宙。宇宙A 是维持现状继续打工，宇宙B 是果断辞职全力创业，宇宙C 是边上班边副业的折中路线，宇宙D 是加入创业公司的创新路径。

每张卡片都不是简单的文字描述，而是一套完整画像——未来图景描绘十年后的生活状态，时间线列出关键节点，收入曲线展示财务走势，幸福指数量化主观感受。

请看宇宙B 和宇宙D 的对比：前者收入峰值更高但波动大，后者更稳健但天花板略低。用户可以像对比商品一样，把四条人生放在同一张图上比较。

5 深度分析展示 2:45 — 3:15
01操作步骤
  1. 在结果页继续向下滚动，依次展示五大深度分析模块。
  2. 依次经过：AI 决策树 → 风险预警 → 未来来信 → 后悔指数 → 价值雷达。
  3. 在"价值雷达"模块停留，展示六维雷达图的交互效果。
  4. 在"未来来信"模块停留，朗读信件片段增强感染力。
02解说词

仅仅"看见未来"还不够，我们进一步提供深度决策分析引擎。

决策树把复杂决定拆成层层分支，量化每个节点的概率与收益；风险预警提前识别每条路径的潜在风险并给出规避建议；未来来信以十年后自己的口吻写一封信——"亲爱的 28 岁的我，谢谢你当年的勇敢"，用情感共鸣驱动理性决策。

后悔指数量化"如果不做这个选择，未来后悔的可能性"；价值雷达用六维度评估每个宇宙与用户核心价值观的匹配度。这套分析不是冷冰冰的数据，而是有温度、有洞察的决策助手。

ECharts 雷达图六维渲染 决策树分支动画展开 未来来信打字机效果
6 决策体检演示 3:15 — 3:45
01操作步骤
  1. 点击导航栏"决策体检"，或访问 /checkup 进入决策体检页。
  2. 展示页面顶部综合评分（如 78 分）与人格标签（如"理性探索者"）。
  3. 向下滚动，展示六维雷达图：信息收集、逻辑分析、风险意识、长期视角、情绪管理、行动力。
  4. 继续滚动，展示各维度详情与行动建议列表。
02解说词

除了帮用户做单次决策，我们还把决策能力本身量化评估。进入决策体检页，系统会基于用户的历史决策行为，给出一个综合评分和人格画像。

比如这位用户得分 78 分，人格标签是"理性探索者"——逻辑强但行动力偏弱。下方的六维雷达图清晰展示了他在信息收集、风险意识、长期视角等六个维度的强弱分布。

每个维度都配有详细解读和行动建议，告诉用户该补哪块短板。这才是真正的成长闭环——不只帮你做决定，更帮你变成更好的决策者。

03预期效果
综合评分数字滚动动画 六维雷达图 ECharts 渲染 人格标签胶囊展示
7 数据驾驶舱演示 3:45 — 4:15
01操作步骤

02解说词

最后带各位评委看一下平台的数据驾驶舱。这里汇聚了全平台的运行数据。

顶部 5 项核心指标一目了然——总模拟次数、活跃用户、决策准确率、平均幸福提升、覆盖城市数。下方三种图表分别用折线图展示模拟趋势、柱状图展示场景分布、环形图展示用户对不同宇宙的偏好。

最下方是实时活动流，每一位用户的决策动态都会在这里滚动呈现，让平台运营者能实时感知用户需求。所有图表均由 ECharts 驱动，支持交互式缩放与详情查看。
