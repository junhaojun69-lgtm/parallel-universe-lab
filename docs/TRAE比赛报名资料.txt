【项目简介】（100字）
人生平行宇宙实验室是一个 AI 驱动的人生决策模拟与成长平台。用户输入人生参数和正在犹豫的决定，AI 基于 DeepSeek 大模型生成四个平行宇宙的未来推演——安稳、突破、成长、创新，每个宇宙包含未来图景、时间线、收入变化和幸福指数，帮助用户预见不同选择的人生轨迹。

【项目简介】（300字）
87% 的年轻人在面临重大人生决策时感到焦虑——考研还是就业？留在大厂还是出去创业？留在北京还是回老家？这些决定可能改变未来十年的人生轨迹，但做决定之前，永远看不到另一条路上会发生什么。

人生平行宇宙实验室通过 AI 模拟引擎，为用户生成不同选择下的平行宇宙发展轨迹。用户输入年龄、学历、城市、职业、收入、存款和正在犹豫的决定，AI 基于 DeepSeek 大模型生成四个维度的平行宇宙推演：

- 安稳宇宙：延续当前轨迹，稳中求进
- 突破宇宙：坚定执行决定，破局重生
- 成长宇宙：渐进转型路径，平衡中求发展
- 创新宇宙：差异化路线，探索另类可能

每个宇宙包含未来图景描述、5年时间线、收入变化曲线和幸福指数评估。此外还提供 AI 决策树、风险预警、未来来信、后悔指数、价值雷达、案例库、未来采访等 8 项深度分析，以及决策复盘、决策档案、AI 决策体检、AI 人生地图等成长闭环功能。

核心理念：人生没有标准答案，但 AI 可以帮你预见可能。

【创新点】

1. 四重平行宇宙推演引擎
独创"安稳/突破/成长/创新"四维推演模型。基于 DeepSeek 大模型，结合用户人生参数深度推演，每个宇宙生成未来图景、时间线、收入变化与幸福指数。不同于传统决策工具的单线性预测，四维并行推演让用户全面看到不同策略下的人生轨迹。

2. 决策全生命周期闭环
从"推演 → 执行 → 复盘 → 档案 → 体检"完整闭环。30/90/180 天三阶段复盘机制，AI 对比原始预测与实际结果，持续优化决策能力。不止于一次性推演，更陪伴用户决策的完整生命周期。

3. 六维决策能力体检系统
风险承受力、执行力、长期主义、学习力、创新力、成长欲六维度量化评估，生成决策人格标签（探索型/创业型/成长型/稳健型），提供 AI 个性化建议。将抽象的"决策能力"量化为可追踪的成长指标。

4. AI 人生地图可视化
将人生路径抽象为可交互的树状图谱，16 个节点、5 种类型（起点/决策/事件/里程碑/终点），贝塞尔曲线连接，点击展开每个节点的可能未来。让用户直观看到人生选择的分支与可能性。

【技术亮点】

1. DeepSeek 大模型驱动
核心推演引擎基于 DeepSeek 大模型，通过精心设计的 Prompt Engineering，生成结构化的平行宇宙推演数据。后端内置完整 Mock 数据兜底，无 API Key 也能完整体验所有功能。

2. Vue 3 全栈架构
前端 Vue 3 Composition API + Vite 构建，10 个页面路由、7 个核心组件、ECharts 数据可视化、html2canvas + jsPDF 报告导出。后端 Express.js RESTful API，15+ 端点，统一错误处理。

3. 城市数据模型
内置 10 个城市的就业指数、薪资指数、创业指数、生活成本指数、发展指数数据模型，AI 推演结合用户所在城市的实际数据，提供更精准的预测。

4. 玻璃拟态 UI 设计
全站采用 Glassmorphism 设计风格，毛玻璃效果 + 渐变色 + 微动画，深色主题搭配紫/蓝/青/粉四色宇宙主题，视觉体验沉浸感强。

5. 数据驾驶舱
5 项核心指标 + 折线图/柱状图/环形图三种动态图表 + 实时活动流，专为比赛演示设计的数据可视化页面。

6. 比赛演示模式
5 分钟路演专用页面，全屏滚动展示项目价值、核心创新、技术架构和应用场景，内置进度指示器和动画效果。

- 心理咨询辅助：帮助来访者理清决策困惑
