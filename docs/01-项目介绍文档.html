01项目概述

Project Overview

人生平行宇宙实验室是一个基于 AI 大模型的人生决策模拟与成长平台。平台以"平行宇宙"为核心隐喻，将人生中每一个犹豫不决的决策，转化为可被 AI 推演、可视化对比、长期追踪的科学实验。

用户只需输入自身的人生参数（年龄、职业、收入、家庭状况等）和正在犹豫的决定（如"是否跳槽到创业公司""是否回二线城市发展"），平台便会调用 DeepSeek 大模型，在数秒内生成四重平行宇宙的未来推演：

宇宙 A · 安稳

保持现状

不做改变，延续当前人生轨迹，推演维持现状的长期发展走向。

宇宙 B · 突破

执行目标

坚定执行用户的决定，推演该选择带来的直接后果与连锁影响。

宇宙 C · 成长

折中路径

AI 推荐的折中方案，在稳健与进取之间寻找平衡的成长路径。

宇宙 D · 创新

另类视角

AI 推荐的另类视角方案，跳出常规思维框架的创造性人生路径。

每个宇宙输出收入变化、风险指数、幸福指数、成长指数和 AI 建议等核心维度数据，并通过雷达图、趋势图直观对比，帮助用户在决策前"预见"不同选择的人生轨迹，从而做出更加理性、从容的人生决策。

平台不止于一次推演，而是构建了从"推演"到"执行"到"复盘"到"档案"再到"体检"的决策全生命周期闭环，让每一次人生选择都成为可量化、可追溯、可优化的成长资产。

02问题背景

Problem Background

人生充满选择，而每一个重大决策都可能深刻改变一个人的命运轨迹。然而，在面临职业转型、城市迁移、教育规划等关键人生节点时，大多数人依赖的仍然是直觉、他人经验和碎片化信息，缺乏科学的决策辅助工具。

痛点一：决策焦虑普遍存在

据调研，87% 的年轻人在面临重大人生决策时感到焦虑和迷茫。职业选择、城市去留、感情抉择、教育投资……每一个决策都伴随着巨大的不确定性，而人脑天然难以同时权衡多个变量的长期影响，导致决策疲劳和回避心理。

痛点二：传统决策方式缺乏科学依据

传统的决策方式——拍脑袋、抛硬币、问亲友——缺乏对未来可能性的系统性预见能力。即便是"利弊分析法"等经典工具，也无法基于个人真实参数推演出不同选择的长期人生轨迹，更无法量化风险与收益。

痛点三：现有工具无法满足个性化需求

市面上的职业规划工具、心理测评产品大多提供标准化、模板化的建议，无法提供基于 AI 大模型的、个性化的、面向未来的决策推演。用户需要的不是一份通用报告，而是"如果我做了这个选择，我的人生会怎样"的具象化预演。

03解决方案

Solution

人生平行宇宙实验室以 "AI 推演 + 全生命周期闭环" 为核心方法论，将人生决策从"一次性赌博"升级为"可实验、可复盘、可优化的科学过程"。

四重平行宇宙推演

基于 DeepSeek 大模型，平台为用户的每一个决策生成四个维度的未来图景：安稳（保持现状）、突破（坚定执行）、成长（折中路径）、创新（另类视角）。每个宇宙都包含 1 年、3 年、5 年的阶段性推演，覆盖收入、风险、幸福、成长四大核心指标，并通过 ECharts 可视化呈现，让用户一目了然地看到不同选择的长期差异。

决策全生命周期闭环

平台不止于一次推演，而是构建了完整的决策成长闭环，让每一次选择都成为可追踪的成长资产：

用户在决策前进行 AI 推演，决策后按 30 天、90 天、180 天节点进行复盘，所有决策沉淀为个人人生档案，最终通过 AI 决策体检量化个人的决策能力成长曲线。

14 项核心功能覆盖决策全流程

从推演、预警、复盘到导出、体检，平台提供 14 项核心功能，覆盖决策前、决策中、决策后的完整流程，形成一套完整的人生决策辅助系统。

04核心功能

Core Features · 14 项功能覆盖决策全流程

01

四重平行宇宙推演

核心功能。基于 DeepSeek 大模型，为用户决策生成安稳、突破、成长、创新四个维度的未来推演，每个宇宙包含多维指标与阶段性发展预测。

02

AI 决策树

将复杂决策拆解为可视化决策树，展示每个分支的概率与影响路径，帮助用户理清决策逻辑，识别关键变量与隐藏风险。

03

风险预警

AI 自动识别每个平行宇宙中的潜在风险点，按严重程度分级预警，并提供针对性的风险缓释建议，防患于未然。

04

AI 未来来信

以"未来自己写给现在的自己"的书信形式，呈现 5 年后不同选择下的人生感悟，用情感共鸣增强决策的代入感与说服力。

05

AI 案例库

汇聚海量真实人生决策案例，AI 智能匹配与用户处境相似的案例参考，让用户从他人的选择与结果中获得启发。

06

后悔指数

独创量化指标，基于大模型推演评估每个选择在 5 年后可能产生的后悔概率，帮助用户规避"未来后悔"型决策。

07

人生价值雷达

通过 ECharts 雷达图多维度对比四个平行宇宙在收入、成长、幸福、风险、自由度等维度的表现，直观呈现选择的综合价值。

08

AI 未来采访

模拟记者采访"未来自己"的互动场景，用户可向不同宇宙中的未来自我提问，获得第一人称视角的人生反馈与建议。

09

决策复盘（30/90/180天）

在决策执行后的 30 天、90 天、180 天三个关键节点，引导用户对照推演结果进行结构化复盘，校验预测准确度并沉淀经验。

10

决策档案

所有决策推演、执行、复盘数据自动归档为个人人生决策档案，形成可追溯的决策历史时间线，构建个人成长数据库。

11

AI 人生地图

将用户所有决策节点串联为可视化人生地图，呈现人生轨迹的全局视图，帮助用户发现决策模式与成长脉络。

12

AI 决策体检

定期对用户的决策历史进行 AI 综合评估，量化决策能力成长曲线，识别决策盲区，生成个性化决策能力提升建议。

13

PDF 报告导出

一键将平行宇宙推演结果、雷达图对比、AI 建议等导出为精美 PDF 报告，便于保存、分享与深度研读。

14

数据驾驶舱

个人决策数据可视化总览面板，聚合决策次数、复盘完成率、后悔指数趋势、能力成长等核心指标，一目了然掌握决策全貌。

05技术亮点

Technical Highlights

DeepSeek AI 大模型 Vue 3 Composition API ECharts 数据可视化 响应式 + 玻璃拟态 UI

DeepSeek AI 大模型驱动

后端集成 DeepSeek 大语言模型 API，通过精心设计的 Prompt 工程将用户人生参数与决策意图结构化输入，引导模型生成包含多维度指标、阶段性预测和个性化建议的四重宇宙推演结果。系统设计了完善的降级策略——在未配置 API Key 时自动切换至 Mock 数据模式，确保产品功能在任何环境下均可完整体验。

Vue 3 Composition API

前端采用 Vue 3 Composition API 构建组件逻辑，通过 setup 函数与响应式 API（ref、reactive、computed）实现细粒度的状态管理。组件设计遵循单一职责原则，UniverseCard、雷达图、趋势图等核心组件高度可复用，代码结构清晰、易于维护与扩展。

ECharts 数据可视化

集成 ECharts 图表库，将 AI 推演的多维数据转化为雷达图、趋势折线图、柱状对比图等直观可视化形态。图表采用与产品一致的深色主题定制，支持交互式数据探索，让用户在视觉层面快速理解不同选择的人生差异。

响应式设计 + 玻璃拟态 UI

采用宇宙主题视觉设计，深色星空背景配合紫色渐变主色调，营造沉浸式探索氛围。全局运用玻璃拟态（Glassmorphism）设计语言——半透明卡片、柔和模糊、微光边框——兼具科技感与美感。响应式布局确保在桌面、平板、移动端均有出色的视觉与交互体验。
