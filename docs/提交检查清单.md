## 部署命令

### GitHub 部署
```powershell
.\deploy-github.ps1 -Token "你的GitHub_Token"
```
Token 获取：https://github.com/settings/tokens（需要 repo 权限）

### Netlify 认领（60分钟内）
打开以下链接认领站点：
https://app.netlify.com/drop/euphonious-baklava-4603ea

### 重新部署到 Netlify
```powershell
cd frontend
npm run build
cd ..
npx netlify deploy --dir ./frontend/dist --prod --allow-anonymous
```
