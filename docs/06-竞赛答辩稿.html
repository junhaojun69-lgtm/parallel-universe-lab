人生平行宇宙实验室

AI 人生决策成长平台

路演演讲 5 分钟 评委答辩 3 分钟 总计 8 分钟

1开场白

【开场】30 秒

尊敬的各位评委老师，大家好！

我们是人生平行宇宙实验室团队。今天，我们为大家带来一款用 AI 帮你预见不同选择下人生轨迹的决策成长平台。

每个人的人生都站在无数个分岔路口，而每一次选择，都可能通向一个截然不同的未来。我们的产品，就是要把那些"看不见的未来"，变成看得见、比得了、能复盘的平行宇宙。

接下来，请允许我用五分钟时间，带各位评委走进这个属于决策者的平行宇宙。

2问题引出

【问题引出】45 秒

在正式展示产品之前，我想先问在座的各位评委老师一个问题：

"如果当初选择了另一条路，你的人生会怎样？"

这个问题，几乎每个人都问过自己。根据我们的调研数据，87% 的年轻人在面临重大人生决策时感到焦虑和迷茫——考研还是就业？留在大城市还是回老家？跳槽转型还是坚守现状？

而更深层的问题是，他们面临三大痛点：

  • 缺乏科学工具——决策往往靠直觉、靠经验、靠"问身边人"，没有系统化的分析手段；
  • 无法预见未来——每条路只能走一次，选错了无法重来，未来是黑箱；
  • 决策后无法复盘——做了选择就翻篇，没有沉淀、没有成长、没有迭代。

正是看到了这个痛点，我们决定用 AI 来解决这个问题——让每一个普通人，都能拥有一个属于自己的"人生推演引擎"。

3产品展示

【产品展示】2 分钟

接下来是今天最核心的部分——我们的产品到底能为用户做什么。

人生平行宇宙实验室的核心功能，是四重平行宇宙推演。用户只需输入自己的人生参数——年龄、职业、收入、家庭状况，以及那个让他犹豫不决的决定，AI 就会在几秒钟内，为他生成四个平行宇宙的未来发展轨迹。

宇宙A · 安稳现状 宇宙B · 突破执行 宇宙C · 成长折中 宇宙D · 创新路径

每一个宇宙都不是简单的文字描述，而是一套完整的未来画像：包含未来图景、十年时间线、收入变化曲线和幸福指数评估。用户可以像对比商品一样，把四条人生路径放在同一张雷达图上，一目了然地看到每条路的风险、收益、成长性和幸福感。

【深度分析】延续展示

但仅仅"看见未来"还不够。我们进一步提供了深度决策分析引擎，包含五大模块：

  • 决策树分析——把复杂决定拆解为层层分支，量化每个节点的概率与收益；
  • 风险预警系统——提前识别每条路径的潜在风险，给出规避建议；
  • 未来来信——以十年后自己的口吻写一封信，用情感共鸣驱动理性决策；
  • 后悔指数——量化"如果不做这个选择，未来后悔的可能性"；
  • 价值雷达——六维度评估每个宇宙与用户核心价值观的匹配度。

这套分析不是冷冰冰的数据，而是有温度、有洞察的决策助手。

【成长闭环】延续展示

更关键的是，我们打造了一个决策全生命周期成长闭环。市面上大多数工具只管"帮你做决定"，做完就结束了。而我们认为，真正的成长发生在决策之后。

因此我们设计了完整的复盘体系：

  • 30 / 90 / 180 天复盘机制——在关键节点提醒用户回看当初的推演，对比现实与预测的偏差；
  • 决策档案——每一次决策自动归档，形成个人专属的决策历史库；
  • 决策体检——定期分析用户的决策模式，发现"总是过于保守"或"容易冲动"等倾向；
  • 人生地图——把所有决策串联成一张可视化的人生轨迹地图，看见自己的成长脉络。

最后，我们还配备了数据驾驶舱，用 ECharts 实时可视化所有决策数据，让用户对自己的"决策人生"拥有上帝视角。

4技术创新

【技术创新】1 分钟

在技术层面，我们有四个核心创新点。

第一，DeepSeek AI 大模型驱动。我们接入了国产领先的 DeepSeek 大模型作为推演引擎，它具备强大的逻辑推理和场景模拟能力，能够基于用户输入的参数，生成符合现实逻辑的未来推演，而非随机编造。

第二，四维推演模型独创。我们自研了"安稳-突破-成长-创新"四维推演框架，这不是简单的正反两面，而是覆盖了保守、激进、折中、创新四种典型决策心理，是目前市面上独有的多维推演方案。

第三，决策全生命周期闭环。从推演、分析、决策、复盘到成长，我们实现了业内首个完整的决策闭环系统，让工具从"一次性咨询"升级为"终身成长伙伴"。

第四，六维决策能力量化。通过价值雷达和决策体检，我们首次将人的"决策能力"变成可量化、可追踪、可提升的指标体系。

技术栈方面，我们采用前后端分离架构：

Vue 3 Express ECharts MySQL DeepSeek API

前端用 Vue 3 保证交互体验，后端用 Express 提供稳定 API，ECharts 负责数据可视化，MySQL 持久化用户决策档案。整套架构轻量、高效、可扩展。

5社会价值与商业价值

【价值阐述】45 秒

一款好的产品，既要创造社会价值，也要具备商业可持续性。

在社会价值方面，我们帮助年轻人从"凭感觉决策"走向"科学化决策"，从根本上缓解决策焦虑。每一次优质的决策，都可能改变一个人的人生轨迹，而千千万万个更好的决策，汇聚起来就是整个社会人力配置效率的提升。我们相信，让决策更理性，就是让社会更高效。

在商业价值方面，我们规划了三层商业模式：

  • B2C 订阅服务——面向个人用户的基础免费 + 高级功能订阅，降低门槛，培养习惯；
  • B2B 企业服务——为企业员工提供职业发展决策支持，作为 HR 福利或 EAP 服务采购；
  • B2G 高校合作——与高校就业指导中心合作，为毕业生提供科学择业工具。

应用场景覆盖广泛：毕业生择业、职场转型、城市迁移、教育规划、创业抉择……凡是涉及人生重大选择的场景，都是我们的用武之地。

7预设答辩问题

以下为评委可能提出的常见问题及预设回答，供答辩环节参考。

Q AI 推演的准确性如何保证？毕竟人生是不可预测的。
A 这是一个非常关键的问题。首先我们要明确，我们的产品不是"算命"，而是"推演"。AI 推演基于用户输入的真实人生参数和 DeepSeek 大模型的逻辑推理能力，生成的是"概率性的未来趋势"，而非确定性的预言。其次，我们通过四维并行推演，覆盖了多种可能性，用户看到的不是单一答案，而是一个"可能性区间"。最后，我们的复盘机制会持续校准推演准确度——用户在 30/90/180 天后回看，系统会对比现实与预测的偏差，不断优化模型。准确性不是一蹴而就的，而是在闭环中持续提升的。
Q 与市面上其他决策工具或 AI 对话产品的区别是什么？
A 区别主要体现在三点。第一，结构化 vs 自由对话。直接问 AI 只能得到泛泛的建议，而我们将决策过程结构化为"四维推演 + 五大分析 + 成长闭环"，输出的是体系化的决策报告。第二，多维度可视化。我们用雷达图、趋势图、决策树等可视化手段呈现结果，而非纯文字。第三，全生命周期陪伴。市面工具大多是一次性咨询，而我们提供从决策前到决策后 180 天的完整复盘体系，这是独有的。
Q 如何实现商业化？用户愿意为这个产品付费吗？
A 我们采用"免费引流 + 订阅增值 + B端拓展"的路径。C 端基础推演免费，降低获客门槛；深度分析、决策体检、人生地图等高级功能按月订阅。B 端面向企业 HR 和高校就业中心，按席位采购。付费意愿方面，决策是人生高频刚需场景，87% 的年轻人有决策焦虑，愿意为"减少一次错误决策"付费——而一次错误决策的代价远高于一年订阅费，这个价值主张是成立的。
Q 用户输入了大量人生隐私数据，数据隐私如何保护？
A 隐私保护是我们的底线。技术上，用户数据采用加密存储，传输全程 HTTPS；架构上，用户可选择匿名模式，无需实名即可使用核心功能；策略上，我们遵循最小化采集原则，只收集推演必需的参数，且用户有权随时删除全部数据。未来我们还将引入本地化推理方案，敏感数据不出端。数据属于用户，我们只是保管者。
Q 技术壁垒在哪里？别人接入 DeepSeek 不也能做吗？
A 大模型是基础设施，壁垒不在模型本身，而在模型之上的领域工程。我们的壁垒有三层：第一，四维推演框架——这是经过大量决策心理学研究提炼的独创方法论，需要持续迭代；第二，Prompt 工程与参数体系——如何把用户的人生参数转化为大模型能理解的推演指令，这里有大量隐性 know-how；第三，决策数据飞轮——随着用户复盘数据积累，我们的推演准确度会持续提升，形成数据壁垒，这是后来者短期无法复制的。
Q 用户留存如何解决？决策不是低频需求吗？
A 表面看决策是低频的，但我们的复盘机制天然创造了留存节点。用户做完一次决策，30 天、90 天、180 天都会收到复盘提醒，这就把一次性行为变成了持续互动。此外，决策体检、人生地图等功能让用户定期回来看自己的"决策成长曲线"，形成习惯。从更高频的角度，我们未来会加入"每日决策小练习""决策案例库"等内容，把低频刚需转化为中频成长陪伴。
Q 团队背景如何？为什么你们能做成这件事？
A 我们团队兼具技术能力和产品洞察。技术侧，团队成员熟悉全栈开发，有 Vue 3 + Express + AI API 的完整工程经验，产品已跑通从前端到后端到 AI 推理的全链路。产品侧，我们深刻理解年轻人的决策焦虑——因为我们自己就是目标用户。我们相信，最好的产品来自最痛的痛点，而我们每天都在用自己的产品做决策。技术可学，但对痛点的真实理解和持续打磨产品的耐心，是我们的核心优势。
Q 未来发展规划是什么？
A 我们规划了三个阶段。近期（6 个月内），完善核心推演和分析功能，积累首批种子用户，验证产品价值假设。中期（6-18 个月），上线订阅体系，拓展 B 端企业合作和高校合作，构建决策数据飞轮，提升推演准确度。远期（18 个月以上），打造"个人决策操作系统"，接入更多人生数据源，成为每个人终身信赖的决策伙伴。我们的愿景是：让科学决策成为每个人的基本能力。
