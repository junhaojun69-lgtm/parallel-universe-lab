问题背景
如果当初做了另一个选择，你的人生会怎样？
内容要点
  • 核心提问：以大字形式抛出"如果当初做了另一个选择，你的人生会怎样？"引发共鸣
  • 关键数据：87% 的年轻人在面临重大决策时感到焦虑与迷茫
  • 痛点一：缺乏科学工具 —— 凭直觉决策，缺少系统化分析手段
  • 痛点二：无法预见未来 —— 选择前看不到不同路径的长期影响
  • 痛点三：决策后无法复盘 —— 选完即结束，缺乏反馈与成长机制
页面顶部以超大字号呈现核心提问，配合渐变色突出；中部用三张数据卡片横向排列展示三大痛点，每张卡片配图标与简短说明；左下角放置 87% 大数字数据可视化，强化问题严重性。

04
核心功能
四重平行宇宙推演
内容要点
  • 安稳宇宙：延续当前轨迹 —— 维持现状，稳步发展，风险最低
  • 突破宇宙：坚定执行决定 —— 全力推进既定选择，追求突破
  • 成长宇宙：渐进转型路径 —— 分阶段过渡，平衡风险与收益
  • 创新宇宙：差异化路线 —— 探索非常规路径，寻找新可能
  • 对比维度：四条路径在收入、成长、风险、幸福感等维度横向对比
四宫格宇宙卡片布局：每个宇宙用独立色系（紫/蓝/青/绿）区分，配专属图标与一句话描述；卡片内展示该宇宙的关键指标预览；四宫格整体呈对称排列，中心可放置"你"的图标作为决策原点。

内容要点

核心功能
决策成长闭环系统
内容要点
  • 闭环五步：推演 → 执行 → 复盘 → 档案 → 体检，形成完整循环
  • 三阶段复盘：30 天 / 90 天 / 180 天分阶段回顾决策效果
  • 决策档案：时间轴形式记录每一次决策的完整过程与结果
  • AI 决策体检：六维能力评估，量化个人决策能力成长
  • AI 人生地图：可视化路径树，呈现人生决策的全局脉络
闭环流程图：以环形布局展示五步骤，箭头首尾相连形成闭环；环内中心放置"成长"核心词；右侧或下方配人生地图树状图预览，展示节点分支与时间轴档案样例。

技术架构
系统技术架构
内容要点
  • 前端层：Vue 3 + Vite + ECharts，响应式界面与数据可视化
  • 后端层：Express.js + RESTful API，提供稳定服务接口
  • AI 层：DeepSeek 大模型 + Prompt Engineering，驱动智能分析
  • 数据层：MySQL 持久化存储 + localStorage 本地缓存
  • 架构特点：前后端分离，模块化设计，易于扩展与维护
分层架构图：自上而下分为前端层、后端层、AI 层、数据层四个色带；每层标注技术栈与核心职责；层间用箭头表示数据流向；右侧可附技术选型理由简表。
