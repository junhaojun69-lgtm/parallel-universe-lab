人生平行宇宙实验室是一个 AI 驱动的人生决策模拟平台。用户输入年龄、学历、城市、职业等人生参数和正在犹豫的决定，系统基于 DeepSeek 大模型生成安稳、突破、成长、创新四个平行宇宙的未来推演，每个宇宙包含未来图景、时间线、收入变化和幸福指数，帮助用户科学预见不同选择的人生轨迹。

300字简介
约300字

87%的年轻人在面临重大人生决策时感到焦虑。考研还是就业？留在大厂还是出去创业？留在北京还是回老家？这些决定可能改变未来十年的人生轨迹，但做决定之前，永远看不到另一条路上会发生什么。

人生平行宇宙实验室用 AI 解决这个问题。用户输入七项人生参数和正在犹豫的决定，系统基于 DeepSeek 大模型，推演出四个平行宇宙的未来图景：安稳宇宙延续当前轨迹，突破宇宙果断执行决定，成长宇宙渐进转型，创新宇宙探索差异化路线。每个宇宙包含五年时间线、收入变化曲线和幸福指数评估。

不止于推演。产品还提供 AI 决策树、风险预警、未来来信、后悔指数、价值雷达等八项深度分析，以及 30/90/180 天三阶段决策复盘、决策档案、六维决策能力体检和 AI 人生地图可视化，覆盖从推演到成长的决策全生命周期。共 14 项核心功能，10 个页面路由，15 个 API 端点。

05
1000字项目介绍
约1000字

人生没有标准答案，但 AI 可以帮你预见可能。这是人生平行宇宙实验室的核心理念。我们构建了一个基于大模型推理的人生决策模拟平台，让用户在做出重大决定之前，提前"体验"不同选择可能带来的未来。

问题背景。调研显示，87%的年轻人在面临重大人生决策时感到焦虑。这种焦虑的根源在于信息不对称：你无法同时走两条路，也无法预知另一条路的终点。传统的决策方式——问朋友、查资料、列优缺点——都停留在经验层面，缺乏个性化和前瞻性。市面上已有的职业规划工具，大多只提供一份性格测试报告，告诉你"适合做什么"，但人生不是性格测试，人生是一个充满可能性的平行宇宙。

解决方案。用户在创建页面输入七项人生参数（年龄、学历、城市、职业、月收入、存款、决策内容），系统调用 DeepSeek 大模型，生成四个平行宇宙的推演结果。安稳宇宙模拟延续当前轨迹的稳中求进路径；突破宇宙模拟果断执行决定的破局重生路径；成长宇宙模拟渐进转型的平衡发展路径；创新宇宙模拟差异化路线的探索路径。每个宇宙包含完整的未来图景描述、五年时间线、收入变化曲线和幸福指数评估，用户可以直观对比四条路径的差异。

深度分析。四重宇宙推演只是起点。系统还会基于推演结果生成八项深度分析：AI 决策树用树状图展示未来可能路径和概率；风险预警分析可能后悔的关键节点；AI 未来来信让三年、五年、十年后的自己写信给现在的你；后悔指数量化不行动的代价；人生价值雷达从财富、健康、家庭、自由、成长、幸福六个维度对比四个宇宙；AI 案例库匹配相似用户案例；AI 未来采访以问答形式呈现未来视角；PDF 报告支持一键导出完整结果。

决策成长闭环。很多决策工具的通病是推演完就结束了。人生平行宇宙实验室做了完整的生命周期闭环：推演、执行、复盘、档案、体检。系统在 30 天、90 天、180 天三个节点提醒用户复盘，填写实际发生的情况，AI 对比原始预测与实际结果，分析偏差原因。所有历史决策进入决策档案，用时间轴呈现。AI 决策体检从风险承受力、执行力、长期主义、学习力、创新力、成长欲六个维度量化评估决策能力，生成决策人格标签。AI 人生地图将人生路径画成一棵可交互的树，16 个节点、5 种类型，点击展开每个节点的可能未来。

技术实现。前端采用 Vue 3 Composition API + Vite 构建，ECharts 负责数据可视化，html2canvas + jsPDF 支持报告导出。后端基于 Express.js 提供 15 个 RESTful API 端点，DeepSeek 大模型通过 Prompt Engineering 生成结构化推演结果，MySQL 存储历史记录，localStorage/sessionStorage 实现跨页传参。UI 采用玻璃拟态设计，深色宇宙主题，响应式布局适配移动端。还内置了数据驾驶舱和 5 分钟路演演示模式，专为比赛场景设计。

06
创新点
四大创新

四重平行宇宙推演引擎

独创"安稳/突破/成长/创新"四维推演模型。基于 DeepSeek 大模型，结合用户人生参数深度推演，每个宇宙生成未来图景、五年时间线、收入变化与幸福指数。市面上的决策工具多为单一维度建议，本产品提供四个维度的平行未来对比。

决策全生命周期闭环

从"推演到执行到复盘到档案到体检"完整闭环。30/90/180 天三阶段复盘机制，AI 对比原始预测与实际结果，分析偏差原因。其他工具只做推演这一步，本产品陪伴决策全生命周期。

六维决策能力体检系统

风险承受力、执行力、长期主义、学习力、创新力、成长欲六维度量化评估，生成决策人格标签（探索型/创业型/成长型/稳健型）。将"决策能力"从抽象概念变为可测量、可追踪的六维指标。

AI 人生地图可视化

将人生路径抽象为可交互的树状图谱，16 个节点、5 种类型（起点/决策/事件/里程碑/终点），贝塞尔曲线连接，点击展开每个节点的可能未来。用户可以像探索游戏一样，探索自己人生的所有可能性。

07
技术亮点
技术栈

前后端分离架构，AI 大模型驱动核心业务逻辑。

Vue 3 Vite Vue Router ECharts Express.js DeepSeek MySQL html2canvas jsPDF Axios
AI 推演引擎：通过 Prompt Engineering 让 DeepSeek 大模型生成结构化 JSON 数据，包含四个宇宙的完整推演。设计了容错机制，API 不可用时自动降级到 Mock 数据，保证 UI 功能完整可用。
前端工程化：Vue 3 Composition API 实现逻辑复用，Vite 提供开发热更新和生产构建。ECharts 渲染折线图、柱状图、环形图、雷达图四种图表。玻璃拟态 UI 配合 CSS 动画系统，深色宇宙主题贯穿全局。
数据流设计：sessionStorage 实现跨页面数据传递（创建页到结果页），localStorage 持久化用户偏好。MySQL 存储历史决策记录，支持时间轴回溯。15 个 RESTful API 端点覆盖全部业务场景。
生产部署：后端自动托管前端 dist 静态文件，单端口提供前后端服务。SPA 路由回退保证刷新不 404。支持 PM2 进程守护和 Nginx 反向代理，可一键部署到云服务器。

社会价值
三个维度

87%的年轻人在面临重大人生决策时感到焦虑。这种焦虑不只是心理问题，它导致拖延、错失机会、甚至抑郁。人生平行宇宙实验室从三个维度创造社会价值。

减少决策焦虑。当用户看到四个平行宇宙的推演结果，焦虑感会显著降低——不是因为 AI 给了"正确答案"，而是因为不确定性被具象化了。看得见的可能性，比看不见的未来更容易接受。
量化决策能力。六维决策能力体检把"决策能力"这个抽象概念变成了可测量、可追踪的指标。用户可以知道自己哪个维度强、哪个维度弱，有针对性地提升。这不是性格测试，是基于真实决策行为的量化评估。
AI 民主化。大模型推理能力不应该只服务于大企业。人生平行宇宙实验室让每个普通人都能获得 AI 辅助决策的能力——9.9 元/月，低于一杯咖啡。这是 AI 技术普惠的一个具体落地。
