项目简介Project Introduction

一句话介绍

基于 DeepSeek 大模型的 AI 人生决策模拟与成长平台，帮你在做决定前预见四重平行宇宙的未来。

100 字简介

人生平行宇宙实验室是一个 AI 驱动的人生决策模拟平台。用户输入人生参数和犹豫的决定，AI 生成四个平行宇宙的未来推演，帮助预见不同选择的人生轨迹。

300 字简介

人生平行宇宙实验室是一个 AI 驱动的人生决策模拟与成长平台。用户输入年龄、学历、城市、职业、收入、存款和正在犹豫的决定，AI 基于 DeepSeek 大模型生成四个维度的平行宇宙推演：安稳宇宙、突破宇宙、成长宇宙、创新宇宙。每个宇宙包含未来图景、5 年时间线、收入变化曲线和幸福指数评估。平台提供 8 项 AI 深度分析和决策全生命周期闭环。

4

创新点Innovation Highlights

01
四重平行宇宙推演引擎
基于 DeepSeek 大模型，将单一决策推演为安稳、突破、成长、创新四个维度的平行宇宙未来。
02
决策全生命周期闭环
从决策输入、推演分析到复盘成长，构建完整的决策记录与回顾闭环体系。
03
六维决策能力体检系统
多维度量化评估用户的决策能力画像，让成长轨迹可视化、可追踪。
04
AI 人生地图可视化
用 ECharts 将人生轨迹、收入曲线与幸福指数渲染为沉浸式可视化地图。
5

技术亮点Technical Highlights

前端
Vue 3 Composition API + Vite + ECharts + 玻璃拟态 UI
采用 Vue 3 组合式 API 与 Vite 构建工具链，ECharts 渲染数据可视化，玻璃拟态风格打造科技感界面。
后端
Express.js + 15 个 RESTful API 端点
基于 Express.js 构建后端服务，提供 15 个标准化 RESTful API 端点，覆盖推演、分析、记录全流程。
AI
DeepSeek 大模型 + Prompt Engineering + 结构化 JSON 输出
深度集成 DeepSeek 大模型，通过精心设计的 Prompt 工程实现稳定结构化 JSON 输出。
数据
MySQL + localStorage 双存储方案
服务端 MySQL 持久化与客户端 localStorage 本地缓存相结合，兼顾数据可靠性与离线体验。
部署
Vercel 全栈部署 + GitHub 版本管理
Vercel 一键全栈部署，GitHub 进行版本管理与持续交付，实现现代化 DevOps 流程。
容错
Mock 数据兜底，无 API Key 也能完整体验
内置 Mock 数据兜底机制，在无 API Key 或服务异常时仍可完整体验全部功能。
