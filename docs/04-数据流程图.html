人生平行宇宙实验室

数据流程图文档

本文档面向比赛评委，系统展示「人生平行宇宙实验室」四条核心数据流程的完整链路。每条流程以独立区域呈现，标注关键节点、技术实现细节与数据流向。

系统基于 Vue 3 前端 + Node.js/Express 后端架构，核心 AI 能力由 DeepSeek 大模型驱动，前端数据通过 sessionStorage 进行页面间传递，可视化图表使用 ECharts 渲染。

普通节点 — 常规处理步骤
AI 节点 — DeepSeek 大模型调用（发光）
存储节点 — 数据存储 / 读取（圆柱形）
箭头 — 数据流向
流程 1

用户输入 → AI 推演 → 结果展示

用户在创建页输入人生参数，系统调用 DeepSeek AI 生成四重平行宇宙推演结果并渲染展示。

Vue 3 Axios Express DeepSeek API sessionStorage
1
用户输入参数
创建页表单：年龄、学历、城市、职业、月收入、存款、决策内容
2
Axios POST 请求
前端通过 Axios 发送 POST 请求至后端 API
3
后端组装 Prompt
拼接系统提示词 + 用户参数，构建完整 Prompt
AI
4
DeepSeek 大模型推演
调用 DeepSeek API，执行四重平行宇宙推演
AI
5
返回四重宇宙结果
安稳 / 突破 / 成长 / 创新 四条平行宇宙路径
6
后端解析 JSON
解析 AI 返回的 JSON，格式校验后返回前端
7
前端渲染结果页
宇宙卡片、决策树、风险预警、未来来信
DATA
8
sessionStorage 存储
结果存入 sessionStorage，供后续页面使用
流程 2

决策复盘流程

用户从档案页选择历史决策，填写实际发生情况后，AI 对比原始预测与实际结果，生成复盘报告。

Vue 3 Express DeepSeek API 决策档案
1
选择历史决策
用户从档案页浏览并选择已有决策记录
2
选择复盘阶段
30 天 / 90 天 / 180 天三个复盘周期
3
填写实际情况
记录实际发生的情况和个人真实感受
AI
4
对比预测与实际
AI 分析原始预测与实际结果的差异
5
生成复盘报告
输出复盘报告，更新决策档案
流程 3

决策体检流程

系统读取用户模拟数据，调用 AI 生成六维决策评估，前端以雷达图与维度卡片展示。

Vue 3 ECharts Express DeepSeek API sessionStorage
DATA
1
读取模拟数据
从 sessionStorage 读取用户模拟数据
2
调用 /api/checkup
前端发送体检请求至后端接口
AI
3
六维评估生成
AI 基于用户决策行为生成六维评估
4
返回评估结果
综合评分、人格标签、维度详情、优劣势、行动建议
5
前端渲染展示
雷达图 + 维度卡片 + 建议列表
流程 4

数据驾驶舱流程

前端请求驾驶舱数据接口，后端返回 Mock 统计数据，前端使用 ECharts 渲染多种图表。

Vue 3 ECharts Express Mock Data

调用 /api/dashboard

3
ECharts 渲染图表
折线图、柱状图、环形图可视化展示
